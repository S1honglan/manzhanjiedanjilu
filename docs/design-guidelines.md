# 设计规范

## 视觉风格

- **风格**：简洁、直观、低饱和色系
- **主色**：淡蓝色系
- **辅色**：灰蓝、米白、浅灰
- **留白**：保持较多留白，避免拥挤
- 整体给人干净、安静、专业的感受

## 颜色规范

### 基础色

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#d8e9f8` | 淡蓝，用于主按钮、标题栏 |
| 主色加深 | `#b3d4f0` | 悬停/选中状态 |
| 背景色 | `#f7f9fc` | 页面整体背景 |
| 卡片背景 | `#ffffff` | 项目卡片、订单卡片 |
| 辅助文本 | `#6f7f97` | 次要文字说明 |
| 主文本 | `#2c3e50` | 标题及正文 |
| 边框 | `#d9e3ed` | 分割线、卡片边框 |

### 订单状态标签颜色

| 状态 | 背景色 | 文字色 | 视觉效果 |
|------|--------|--------|----------|
| `预付定金` | `#cce0f8` | `#3a6d99` | 淡蓝色标签 |
| `工作中` | `#fde4cc` | `#8a5d3d` | 淡橙色标签 |
| `交付` | `#ccf0db` | `#3d7a5a` | 淡绿色标签 |

## 排版规范

### 字体层级（订单卡片内）

| 元素 | 字号 | 字重 | 说明 |
|------|------|------|------|
| 角色名（大标题） | `20px-24px` | `700` | 卡片最醒目的文字 |
| 客户昵称 | `14px-16px` | `500` | 次要标识 |
| 其他字段标签 | `12px` | `400` | 小字灰色标签 |
| 其他字段值 | `14px` | `500` | 小字对应数值 |
| 备注 | `12px` | `400` | 最小字，可折叠 |

### 通用排版

- 行高：1.5
- 字族：系统默认（`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei"`）

## 卡片组件规范

### 项目卡片

```
┌──────────────────────────┐
│ 漫展名称            ✎ × │
│ 漫展日期 · 3个订单       │
└──────────────────────────┘
```

- 圆角：12px
- 阴影：`0 2px 8px rgba(0,0,0,0.06)`
- 内边距：16px-20px
- 悬停时阴影加深 + 轻微上移

### 客户订单卡片

```
┌──────────────────────────┐
│ ★ 角色名（大标题）         │
│ 客户昵称                  │
│                          │
│ 张数: 120张   全款: ¥880 │
│ 定金: ¥400   收入: ¥880  │
│                          │
│ [预付定金]        备注 ▼  │
└──────────────────────────┘
```

- 圆角：10px
- 阴影：`0 1px 4px rgba(0,0,0,0.05)`
- 内边距：14px-16px
- 角色名左对齐，占据卡片上方

## 按钮规范

| 类型 | 样式 |
|------|------|
| 主操作按钮 | 淡蓝填充 `#d8e9f8`，深色文字，圆角 8px |
| 次要按钮 | 淡蓝边框，透明背景，文字同色 |
| 危险按钮 | 淡红文字，悬停淡红背景（删除用） |
| 图标按钮 | 小号文字，透明背景（编辑 ✎、删除 ×） |

## 布局规范

### 桌面端（≥768px）

- 项目卡片网格：2-3 列
- 订单卡片网格：2 列
- 顶部导航栏固定
- 最大内容宽度：960px，居中

### 手机端（<768px）

- 单列卡片列表
- 底部导航栏固定（首页 / 新增 / 搜索）
- 筛选器可折叠
- 卡片占满屏幕宽度（留少量边距）

## 交互规范

- 卡片点击进入内页（项目卡片→项目订单页）
- 表单以底部面板或居中模态框呈现
- 删除操作弹出简单确认（"确定删除吗？"）
- 搜索框输入即过滤，实时响应
- 操作完成后给予简单反馈（如时间提示）
